经历相关：从Windows转移到macOS约半年时间。
本文基于从Windows 10 1909切换到macOS Catalina撰写。

你当然可以通过直接安装Windows来解决使用习惯的问题，但是那样就失去了Mac的灵魂，也会导致性能、优化弱鸡。
所以我推荐通过安装第三方软件解决问题。
要解决麻烦，首先要知道区别在哪里。在我体验macOS之后，有比较强烈的缺失感的有：

  • 窗口贴边吸附
  • 剪贴板历史记录
  • 一个「比较好」的压缩软件
  • NTFS读写
  • 开始菜单

于是本文将分为四个部分推荐软件。

1. 窗口贴边吸附

一说到这个，肯定会有人提到Magnet。
但我不推荐它。因为它在App Store要¥12才能使用。

这就非常之难受：难道Windows自带的功能我macOS要用第三方软件不说，还得付钱？
所以我发现了这样一款挺不错的软件：Rectangle
这个软件支持好多种窗口位置，看图：

并且每一个都支持自定义快捷方式：

支持鼠标拖拽贴边隐藏。体验下来基本可以做到Windows用户不别扭、直觉操作的水平。
下载地址：https://github.com/rxhanson/Rectangle/releases/latest

2. 剪贴板历史记录

Windows 10某个版本中，加入了一个功能，叫做剪贴板历史记录：

这个功能非常人性化、方便啊，只需要按下Windows徽标键+V就能呼出剪贴板历史记录，双击直接粘贴，就不用跑来跑去。用过这个功能的人应该都是深有体会。
其实macOS也有这样的第三方软件，叫CopyLess 2。

主界面整体而言比Windows更修长，并且还多了复制来源的软件。在安装助理程序之后，还能支持双击直接粘贴到当前激活的编辑框。
那Pro版本有啥功能呢？

  • iCloud同步
  • 最多保留1000个粘贴条目

其实我觉得性价比并不高。因为我到现在也还没碰到免费版的条目数量限制……
并且还有Windows没有的特色功能：连串拷贝&粘贴。你可以通过CopyLess来不断复制，然后一并粘贴到一个编辑框中。

下载地址：‎‎CopyLess 2

3. 一个「比较好」的压缩软件

Windows下面，有一个非常好用的软件Bandizip。它当然也有macOS版本，不过并不是免费的，似乎费用还挺高。
这里推荐eZip。功能不赘述了，基本与Bandizip齐平。并且支持在压缩时忽略macOS系统临时文件，对Windows用户更友好。
下载地址：eZip - 专为 macOS 而设计的压缩软件！

4. NTFS读写

这里也有几款非常有名的软件，如Paragon、Tuxera之类的软件。但是它们都收费，都收费！
这里推荐Mounty这款体积非常非常小的插件：

完全免费，随用随开。它并没有图形界面，启动后只会在菜单栏显示一个小图标：

然后就可以实现NTFS读写了。非常方便。
安装方法：

  • Homebrew安装：
    brew cask install mounty
  • dmg直接安装：
    https://mounty.app/releases/Mounty.dmg

5. 开始菜单

其实macOS并不是没有开始菜单，只是拆分到了几处：

  • 位于左上角Apple Logo的电源菜单
  • Launchpad
  • 访达边栏的目录快捷方式

然而有没有办法可以合在一起呢？
很遗憾，没有。但是可以稍微曲线救国一下。因为大多数Windows用户打开开始菜单就是奔着应用程序去的，所以我们可以设置一个触发角（系统偏好设置》桌面&屏幕保护程序》屏幕保护程序》触发角），在左下角的地方直接呼出Launchpad。

这样当你鼠标移动到屏幕左下角的时候，就会弹出Launchpad。也算是曲线救国。
