WWDC 2020预热 - 知乎

北京时间2020年6月23日凌晨1时，Apple 将以全新的虚拟形式形式举行 WWDC 活动。
苹果说这将会是迄今为止规模最大的 WWDC ，因为所有开发者都能无门槛地参与。

已经透露的信息和看点

本次大会可能会发布一个小硬件：AirTags

据Novabach透露的图片，AirTags长这样👆。
它是一个苹果很早（iPhone 11致创新发布会上）就想要发布的产品，因为供应链问题，被紧急砍掉了。它依赖UWB工作，配合AR来寻找你附着了AirTags的物件，所以我们会看到iPhone 11 Pro系列上搭载了U1芯片，但是却没有在发布会上提到，唯一的线索只有总结页角落里的一个小小的“UWB支持”。
去年WWDC上发布的全新查找App也许也是为它铺路的行为。

必定要发布的一堆OS

绿钩：看完 WWDC 发现猜对了。

iOS

震惊！这次的iOS 14透露出来的信息真的好少！

  • Apple Pay支持全国交通联合一卡通、支付宝
  • 主屏幕小组件✅
  • iMessage支持撤回、群聊
  • Siri支持语音定制
  • 可能要重新改回iPhoneOS这名字以不那么突兀

macOS

更少。

  • 基于iOS版重制的Catalyst版本iMessage app✅
  • 推进ARM Mac的进展，开始发展ARM软件生态✅
    但我认为，苹果不会用ARM的Mac全面替代现在的Intel Mac。我想会以之前的MacBook，没有后缀的轻薄版本的方式呈现。因为现在ARM的性能都已经达到瓶颈，连上到3GHz以上都非常困难，而Intel i5散热好些睿频马上就3.5GHz。

iPadOS

没什么的，大抵与iOS/iPhoneOS相同就是了。✅

watchOS

这看点挺多的。

  • 使用iCloud共享相册制作表盘✅
  • 类似iOS/iPadOS的Measure测距仪功能
    我很不解啊，依赖加速度传感器？那不会很不明确吗……又没有摄像头……
  • 家长控制（真·大天才电话手表）
  • 睡眠模式，AutoSleep啥的都可以卸了✅

tvOS

还是没啥可说，没什么消息透出来……

大会议程

本次大会，分为5个部分：

  • 特别活动主题演讲：讲解Apple各个平台“令人兴奋”的创新和更新，可以从apple.com、Apple Developer app、Apple Developer 网站、Apple TV app 和 YouTube 观看。对于中国大陆地区的观众，此腾讯视频，爱奇艺，优酷和哔哩哔哩也将播出。
  • Platforms State of the Union：讲解各个OS的最新进展，将会在Apple Developer app 和 Apple Developer 网站以点播形式提供。中国大陆地区的观众也可前往腾讯视频、爱奇艺、优酷和哔哩哔哩观看。
  • 100 多场工程设计讲座：App的技术和设计主题讲座。在 Apple Developer app 以及 Apple Developer 网站上提供。
  • 开发者论坛：全新的 Apple 开发者论坛会在WWDC期间连线超过1000位Apple工程师。Apple Developer Program 会员还可以向工程师提问。
  • 一对一开发者实验室：需预约，将在有名额的情况下向 Apple Developer Program 会员开放。

详细时间可参阅 Apple Newsroom。
